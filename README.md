# AfriShop Django Demo

AfriShop is a Django delivery-app demo with separate customer and seller flows.

## Run locally

```bash
.venv/bin/python manage.py runserver 0.0.0.0:8000
```

## Demo credentials

| Role | Username | Password |
| --- | --- | --- |
| Admin | `admin` | `Admin@123` |
| Customer | `customer` | `Customer@123` |
| Seller | `seller` | `Seller@123` |
| Delivery | `delivery` | `Delivery@123` |

Customer app: `http://127.0.0.1:8000/`

Seller app: `http://127.0.0.1:8000/seller/login/`

Admin backend: `http://127.0.0.1:8000/admin/`

Mobile API: `http://127.0.0.1:8000/api/`

The demo is configured for Ludhiana, Punjab only. Stores include latitude and longitude so the mobile app can open the exact shop location in maps.
